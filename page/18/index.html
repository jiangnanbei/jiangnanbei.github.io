学习和智商有关系吗?

发表于 2017-03-29 | 分类于 心智

相信你们看到这个标题就知道我要说学习和智商是没有关系的,是的,注意哦,我说的是学习.

学习关键是你要知道怎么去学习,否则你只能做一个笨蛋.

作为一个非科班出身的野生程序员,从我开始到现在的自学经验告诉我,学习应该是这样的:首先,你要知道哪里有问题,然后把大问题分解成小问题,最后再一个一个地去解决,最终你肯定会搞明白所有的问题.

学习的关键就是你每天都要进步,哪怕你每天只懂了一点点.要记住:学习不是百米赛跑,学习是看不到尽头的马拉松.你每天都向前走一点,越往前走你会发现你前面的人越少,当你发现你在前面已经看不到人的时候,你就已经非常非常非常厉害了.

《三体》给我的商业灵感

发表于 2017-03-10 | 分类于 商业

在《三体》中出现了这么一个词“降维”，什么意思呢？打个比方吧，既然在这个维度里我打不过你，那我就跳到一个低维度，从侧翼打击你。

这就让我想到了降维化生存的小米，小米手机的出发点是做人人都用的起的手机，在这里小米降维到一个低维度战区，苹果，华为，三星你们都很，牛但是我不跟你们玩，我到一个你们到不了的维度打击你们。

为什么说小米可以降维而苹果，华为，三星却不可以？因为小米实现了零广告费，零库存和零渠道费，这里不是简单的把广告费由多变少，把库存由多变少，把渠道费由多变少，而是几乎变没了，整个维度都没了。

这就叫降维，天经地义应该存在的维度就这么消失了。当一个企业不需要广告费，不需要库存，不需要渠道费就能生存的时候，它转身去攻击那些这些才能生存的企业，这件事情是令人恐怖的。

雷军说:“我们不打价格战，我们直接降到成本价”。（这里的成本价是指BOM定价，即按照物料清单成本定价，不包含固定成本，研发费用和制造费用。BOM定价是一种比我们所理解的成本价还要狠的一种定价方式）。

那么问题来了，在定价降维后的小米如何在这个维度打击对手呢？要知道一个没有利润的企业是不可能存活下去的，但是事实是小米的的利润曾一度超越腾讯。

小米是如何做到的？因为小米有二次打击能力，这就是说，你可以降到一个更低的维度通过免费的模式打击敌人，但是你必须要有价值链的递延。这就是小米成功的秘密，一个利润位居中国互联网企业前三，市值位居互联网企业第四的商业奇迹的诞生的原因。

如何在这个不确定的世界感知对未来世界的那一丝确定性？

发表于 2017-03-05 | 分类于 心智

这个世界正在变得越来越让人看不懂，你真的很难感知未来，但是有一点我们每一个人可以做到，那就是对自己的把握。那我是怎么“感知未来”的呢？

1.我确信未来的五年我会每一天都过得很充实，每一天都有成长。

2.我确信未来的五年我会比现在懂得更多，比现在活得更加通透。

3.我确信未来的五年我会建立自己的企业，她可以不赚很多钱，但是她必须对这个社会有所贡献。

4.我确信未来的五年我会把握的目标一点一点的都实现。

别的我不敢保证，但是以上这四点我真的就可以做到。恩，你应该知道如何在这个不确定的世界感知对未来世界的那一丝确定性了。
