JSD 乐队网

发表于 2017-07-06 | 分类于 编程

项目简介

为某国际知名乐队设计一个网站。首先这个网站必须和这个乐队一样，要酷。但是同时，这个网站必须对残疾用户以及搜索引擎保持友好。

开办这个网站的主要目的就是发布有关乐队的信息。无论怎么构思这个网站，首先都得确保这些信息能让访客一目了然。

原始资料

客户已经提交了构建网站所需的东西：有关乐队的介绍材料、巡演日程，还有一些照片。这个网站不需要太多的页面，他本质上就是一个宣传手册，而这一点也正是你要把握的核心用户体验。

站点结构

说到页面，首先得有一个详细介绍乐队背景信息的页面。其次要有一个类似相册的放照片的页面。巡演日程安排也需要一个单独的页面。为了让歌迷与乐队沟通，还必须有一个练习页面。最后，当然还要有一个主页，放上乐队简介和站点导航信息。以下是要创建的一个页面：

虽然每个页面的内容不一样，但它们都要使用相同的基本结构。下面该考虑为这些页面创建一个模板了。

页面结构

站点的每个页面都要分成几个区域：

在设计好页面结构之后，下面就要一页一页的插入内容了。不过，先让我们来设想一下站点完工之后的外观。

设计

既然知道了每个页面中都包含哪些结构化元素，而且手里也已经有了客户提供的资料，那么接下来的外观设计就不难了。你可以选择 PS、Fireworks 或其他任何图形设计工具，做出你认为适合的任何风格的设计方案。

做完了视觉设计之后，可以把平面设计切分成多个图片。把所有的这些图片都保存到 images 文件夹中。

CSS

现在，你有了基本的 HTML 模板，也知道自己的站点长什么样了。通过为模板应用 CSS ，可以在 WEB 上再现你的设计方案。

如果把 CSS 都放到一个文件中，可能会为后期的维护带来一些麻烦。而把所有 CSS 分别放在几个文件中则是好主意。

怎样组织 CSS 由你决定，但最好用其中一个保存与整体布局有关的样式，用另一个作为专门的颜色样式表，而用第三个来保存于版式有关的样式：

颜色

样式表 color.css 是在最直观的。记住，不管为那个元素应用什么颜色，都要同时给它一个背景色。否则，就有可能导致意外，看不到某些文本。

布局

基本的布局还是相当简单的，所有内容都在一栏中。

为了让导航中的链接水平排列，需要应用一些浮动效果。除此之外， layout.css 也没有什么不好理解的了。

首先是为 HTML5 块元素定义默认的样式。主要针对那些不支持他们的浏览器，好让这些元素都能够具有适当的布局。

其次，要注意 css reset 的编写。
